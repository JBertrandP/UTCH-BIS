# Travel Recommendation Project

Bienvenido al proyecto de recomendación de viajes. Esta aplicación te ayuda a planificar tu próximo viaje proporcionando recomendaciones personalizadas basadas en tus preferencias.

---

## *Descripción del Proyecto*

Este proyecto es una aplicación web que permite a los usuarios seleccionar un destino, interés, presupuesto, clima y época del año. Luego, la aplicación genera recomendaciones personalizadas, muestra imágenes relacionadas y un mapa interactivo del destino seleccionado.

### *Características Principales*
- Selección de destino, interés, presupuesto, clima y época.
- Recomendaciones basadas en información de Wikipedia.
- Galería de imágenes relacionadas con el destino.
- Mapa interactivo con la ubicación del destino.

---

## *Tecnologías Utilizadas*

### *Frontend*
- *HTML*: Estructura de la página web.
- *CSS*: Estilos y diseño.
- *JavaScript*: Lógica del frontend y manejo de eventos.
- *Leaflet.js*: Biblioteca para mostrar mapas interactivos.

### *Backend*
- *PHP*: Lógica del backend y llamadas a APIs externas.
- *APIs Externas*:
  - *Unsplash*: Para obtener imágenes relacionadas.
  - *OpenWeatherMap*: Para obtener coordenadas del destino.
  - *Wikipedia*: Para obtener información y recomendaciones.

---

## *Instalación y Configuración*

### *Requisitos*
- Servidor web local (por ejemplo, XAMPP o WAMP).
- Navegador web moderno (Chrome, Firefox, Edge, etc.).

### *Pasos para Configurar el Proyecto*
1. Clona el repositorio o descarga los archivos del proyecto.
2. Coloca los archivos en la carpeta htdocs de tu servidor local (por ejemplo, C:\xampp\htdocs\travel_recommendation).
3. Inicia el servidor web (por ejemplo, Apache en XAMPP).
4. Abre el navegador y visita http://localhost/travel_recommendation/index.html.

---

## *Uso del Proyecto*

1. *Selecciona un Destino*:
   - Elige un destino de la lista desplegable (por ejemplo, París, Tokio, etc.).

2. *Selecciona tu Interés*:
   - Elige un interés relacionado con el destino (por ejemplo, Arte, Historia, etc.).

3. *Configura Opciones Adicionales*:
   - Selecciona tu presupuesto, clima preferido y época del año.

4. *Haz Clic en "Buscar Destino"*:
   - La aplicación generará recomendaciones, mostrará imágenes y un mapa del destino seleccionado.
